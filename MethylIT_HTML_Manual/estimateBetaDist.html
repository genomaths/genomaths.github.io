.estimateBetaDist {MethylIT}R Documentation

Select the beta distribution that fit specified quantiles

Description

This function perform a rough estimation of the shape parameters of beta distribution

Usage

.estimateBetaDist(q, init.pars = c(1, 1))

Arguments

q

prior probabilities

init.pars

initial parameter values. Defaults to alpha = 1 & beta = 1, which imply the parsimony pseudo-counts greater than zero.

Details

In order to obtain the estimates for shape parameters of beta distribution, the squared of the difference between the empirical cummulative density function (ecdf) & the theoretical cdf is minimized using the Non-Linear Minimization function of 'stats' package.

Value

the estimated values of the shape parameters of the selected beta distribution.
