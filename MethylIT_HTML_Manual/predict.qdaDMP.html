predict.qdaDMP {MethylIT}R Documentation

Predict function for 'qdaDMP' method

Description

Predict using a qda model built with function evaluateDIMPclass

Usage

## S3 method for class 'qdaDMP'
predict(object, newdata, type = c("all", "class",
  "posterior", "scores", "qda.pred"), ...)

Arguments

object

To use with function 'predict'. An object from class 'qdaDMP'

newdata

To use with function 'predict'. New data for classification prediction

type

To use with function 'predict'. The type of prediction required. The default is "all" basic predictions: classes and posterior classification probabilities. Option "qda.pred" returns the object given by function 'predict.qda' from MASS package: 'class', 'posterior', 'scores' (cases scores on discriminant variables, see qda.

...

arguments passed to or from other methods.

estimateCutPoint, qda
